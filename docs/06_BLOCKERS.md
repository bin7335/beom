## 해결된 이슈

### 1. 카카오 로컬 API 인증 오류

- 증상:
  - 최초 실행 시 `401 Unauthorized`
  - 설정 보완 전 `403 Forbidden`
- 원인:
  - REST API 키 및 API 활성화 설정 문제
- 조치:
  - `.env.local`의 `KAKAO_REST_API_KEY` 재설정
  - 카카오 로컬 API 사용 설정 점검 후 재실행
- 결과:
  - 좌표 변환 정상 완료

### 2. 결과 페이지 개념 혼동

- 증상:
  - 차트와 추천 카드가 모두 비슷한 의미처럼 보여 `환경 진단`과 `추천`의 차이가 흐려짐
- 조치:
  - 차트를 `학교 환경 진단`으로 재정의
  - 추천 카드를 `추천 교육 TOP 3`로 분리
  - 환경 점수 방향을 `낮을수록 취약`으로 고정
- 결과:
  - 진단과 제안의 역할 분리가 명확해짐

### 3. 환경 진단과 환경 지표 섹션 분리로 인한 UX 중복

- 증상:
  - `학교 환경 진단`과 `학교 환경 지표`가 비슷한 내용인데 떨어져 있어 사용자 입장에서 중복처럼 보임
- 조치:
  - 환경 지표 4개를 `학교 환경 진단` 카드 안의 `진단 근거 지표`로 통합
- 결과:
  - 차트 요약과 근거 데이터의 관계가 같은 섹션 안에서 바로 이해되도록 개선
