## 3. 좌표 변환 가이드 (NEIS 주소 → 위경도)

NEIS는 주소만 주고 좌표를 안 줍니다. 변환 필요.

### 옵션 1: 카카오 로컬 API (권장)
- https://developers.kakao.com
- 일 30만 건 무료
- 한국 주소 정확도 우수

### 옵션 2: VWorld Geocoder
- https://www.vworld.kr
- 정부 공식. 한국 주소 표준
- 무료

### 옵션 3: 행정안전부 도로명주소 API
- https://business.juso.go.kr
- 공식 주소 변환

### 처리 스크립트 예시 (카카오 기준)
```python
import requests
import pandas as pd
import time

KAKAO_KEY = "YOUR_REST_API_KEY"

def addr_to_coord(addr):
    url = "https://dapi.kakao.com/v2/local/search/address.json"
    headers = {"Authorization": f"KakaoAK {KAKAO_KEY}"}
    params = {"query": addr}
    res = requests.get(url, headers=headers, params=params)
    docs = res.json().get("documents", [])
    if docs:
        return float(docs[0]["x"]), float(docs[0]["y"])  # 경도, 위도
    return None, None

df = pd.read_csv("data/raw/neis_schools.csv")
coords = []
for addr in df["주소"]:
    lng, lat = addr_to_coord(addr)
    coords.append((lng, lat))
    time.sleep(0.1)  # 호출 간격

df["lng"], df["lat"] = zip(*coords)
df.to_csv("data/processed/schools_with_coords.csv", index=False)
```

## 4. 반경 매칭 사전 계산 가이드 (Day 6)

### Haversine 공식으로 거리 계산

```python
import numpy as np

def haversine_km(lat1, lng1, lat2, lng2):
    R = 6371  # 지구 반지름 km
    lat1, lng1, lat2, lng2 = map(np.radians, [lat1, lng1, lat2, lng2])
    dlat = lat2 - lat1
    dlng = lng2 - lng1
    a = np.sin(dlat/2)**2 + np.cos(lat1) * np.cos(lat2) * np.sin(dlng/2)**2
    c = 2 * np.arcsin(np.sqrt(a))
    return R * c

# 학교 1개 기준 반경 1km 안의 데이터 포인트 찾기
for school in schools:
    nearby = []
    for accident in accidents:
        d = haversine_km(school.lat, school.lng, accident.lat, accident.lng)
        if d <= 1.0:
            nearby.append(accident)
    school.nearby_accidents = len(nearby)
```

### 470개 학교 × N개 데이터셋
- 한 데이터셋 1만 건 가정 시: 470 × 10000 = 470만 회 계산
- numpy 벡터화하면 수 초 내 완료

## 5. 데이터 적재 형식 (Supabase)

### schools 테이블
```sql
CREATE TABLE schools (
  code TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  address TEXT,
  lat DOUBLE PRECISION,
  lng DOUBLE PRECISION,
  student_count INT,
  class_count INT,
  sido TEXT DEFAULT '경상북도',
  sigungu TEXT,
  created_at TIMESTAMPTZ DEFAULT now()
);

CREATE INDEX idx_schools_name ON schools USING gin (name gin_trgm_ops);
```

### school_environments 테이블
```sql
CREATE TABLE school_environments (
  school_code TEXT PRIMARY KEY REFERENCES schools(code),
  environment JSONB NOT NULL,
  computed_at TIMESTAMPTZ DEFAULT now()
);

-- environment JSONB 예시
-- {
--   "traffic_accidents_3yr": 18,

```

CREATE TABLE recommendations_cache (
